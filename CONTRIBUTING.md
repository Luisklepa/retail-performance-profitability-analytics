# Contributing

Thanks for your interest. This is a portfolio repository and is intentionally kept small and focused.

## Suggested contributions
- Improvements to documentation (clarity, typos, structure)
- Additional analysis ideas (new measures/visuals) described in issues
- Quality-of-life repo upgrades (templates, organization)

## How to propose an enhancement
1. Open an issue using the **Enhancement** template.
2. Describe the change, the motivation, and what “done” looks like.
3. If you include data exports, place them in `docs/` or `assets/` (the repo ignores exports elsewhere by default).

## Pull requests
- Keep PRs small and scoped.
- Update `README.md` or `docs/` if your change affects usage or interpretation.
- Do not add `.pbix` files to this repository.
